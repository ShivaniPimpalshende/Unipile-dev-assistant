API NAME: Restart an account

METHOD: POST

ENDPOINT:
https://{subdomain}.unipile.com:{port}/api/v1/accounts/{id}/restart

DESCRIPTION:
Restart the sources of a frozen account.

==================================================
REQUEST EXAMPLES
==================================================
