LIST ALL MESSAGES FROM A CHAT
-----------------------------
Method: GET
URL: https://{subdomain}.unipile.com:{port}/api/v1/chats/{chat_id}/messages

DESCRIPTION
-----------
Returns a list of messages related to the given chat. Supports filters like cursor, before/after datetime, limit, and sender_id.

PATH PARAMS
-----------
chat_id
Type: string
Required: yes
Description: The id of the chat related to requested messages.

QUERY PARAMS
------------
cursor
Type: string, length â‰¥ 1
Description: Cursor for pagination. Use value from previous response to fetch next page.

before
Type: string
Description: Filter messages created before this ISO 8601 UTC datetime (exclusive).

after
Type: string
Description: Filter messages created after this ISO 8601 UTC datetime (exclusive).

limit
Type: integer (1-250)
Description: Maximum number of messages to return.

sender_id
Type: string
Description: Filter messages from a specific sender.
