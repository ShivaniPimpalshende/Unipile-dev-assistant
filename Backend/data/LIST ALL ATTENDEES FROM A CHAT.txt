LIST ALL ATTENDEES FROM A CHAT
-------------------------------
Method: GET
URL: https://{subdomain}.unipile.com:{port}/api/v1/chats/{chat_id}/attendees

DESCRIPTION
-----------
Returns a list of messaging attendees related to a given chat. Optional filters can be applied depending on the client implementation.

PATH PARAMS
-----------
chat_id
Type: string
Required: yes
Description: The id of the chat related to requested attendees.
