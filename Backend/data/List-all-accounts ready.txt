API Name: Get Accounts List

Method:
GET

Endpoint:
https://{subdomain}.unipile.com:{port}/api/v1/accounts

Description:
Returns a list of the accounts linked to Unipile.

--------------------------------------------------------------------
QUERY PARAMETERS
--------------------------------------------------------------------
cursor (string, length â‰¥ 1)
A cursor for pagination purposes.
To get the next page of results, make a new request using the cursor
received in the previous response.
Repeat this process until all entries are retrieved.

limit (integer, range: 1 to 250)
Defines the maximum number of items returned in the response.

--------------------------------------------------------------------
SUCCESS RESPONSE
--------------------------------------------------------------------
200 OK
The request succeeded and returns a list of linked accounts.

Response Object:
- object: "AccountList"
- items: array of Account objects

Each Account object may contain:
- id
- name
- type
- created_at
- current_signature
- signatures
- groups
- sources
- connection_params (provider-specific)
- last_fetched_at (if applicable)
- sync_token (for calendar providers)

--------------------------------------------------------------------
ACCOUNT TYPES IN RESPONSE
--------------------------------------------------------------------
Possible account types returned:
- MOBILE
- MAIL
- GOOGLE_OAUTH
- ICLOUD
- OUTLOOK
- GOOGLE_CALENDAR
- WHATSAPP
- LINKEDIN
- SLACK
- TWITTER
- EXCHANGE
- TELEGRAM
- INSTAGRAM
- MESSENGER

Each account type includes provider-specific connection parameters.
