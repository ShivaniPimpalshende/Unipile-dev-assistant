API NAME: List all chats

METHOD: GET

ENDPOINT:
https://{subdomain}.unipile.com:{port}/api/v1/chats

DESCRIPTION:
Returns a list of chats. Optional query parameters can be used
to filter results (unread status, pagination, date range, provider, account).

==================================================
QUERY PARAMETERS
==================================================

unread (boolean)
- true  → return only unread chats
- false → return only read chats

cursor (string, length >= 1)
- Cursor for pagination.
- Use the cursor returned in the previous response to fetch next page.

before (string, ISO 8601 UTC)
- Filter chats created before a datetime (exclusive).
- Format: YYYY-MM-DDTHH:MM:SS.sssZ

after (string, ISO 8601 UTC)
- Filter chats created after a datetime (exclusive).
- Format: YYYY-MM-DDTHH:MM:SS.sssZ

limit (integer, 1–250)
- Maximum number of chats returned.

account_type (string, enum)
- Filter by provider.
- Allowed values:
  WHATSAPP
  LINKEDIN
  SLACK
  TWITTER
  MESSENGER
  INSTAGRAM
  TELEGRAM

account_id (string, length >= 1)
- Filter by account id.
- Can be a comma-separated list of account ids.
