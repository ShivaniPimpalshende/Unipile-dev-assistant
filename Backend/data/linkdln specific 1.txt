API Name: Retrieve Recruiter hiring projects

Method:
GET

Endpoint:
https://{subdomain}.unipile.com:{port}/api/v1/linkedin/projects

Description:
Retrieve list of LinkdIn Recruiter hiring projects.

--------------------------------------------------------------------
QUERY PARAMETERS
--------------------------------------------------------------------
limit
integer
1 to 100
Defaults to 10
A limit for the number of items returned in the response. The value can be set between 1 and 100.

cursor
string
length â‰¥ 1
A cursor for pagination purposes. To get the next page of entries, you need to make a new request and fulfill this field with the cursor received in the preceding request. This process should be repeated until all entries have been retrieved.

sort_order
string
enum
The sorting order for the hiring projects. Default is DESCENDING.
Allowed:
 ASCENDING
 DESCENDING

sort_by
string
enum
The sorting method for the hiring projects. Default is ACCESSED_TIME.
Allowed:
 NAME
 FAVORITE
 CREATED_TIME
 ACCESSED_TIME
 ENGAGED_TIME
 ENGAGEMENT_COUNT

account_id
string
required
The ID of the account to trigger the request from.

--------------------------------------------------------------------
SUCCESS RESPONSE
--------------------------------------------------------------------
200 OK
The request succeeded 

Response Object:
- object: "RecruiterHiringProjectList"
- items: array of Account objects

Each Account object may contain:
- id
- name
- type
- owner_name
- owner_id
- created_at
- job_posting
- sources
- total_count
- cursor
