Forward a message
post
https://{subdomain}.unipile.com:{port}/api/v1/messages/{message_id}/forward
Forward a message to a chat. (Whatsapp only)

Path Params
message_id
string
required
The id of the message to forward.

Body Params
chat_id
string
required
The id of the chat to forward the message to.
